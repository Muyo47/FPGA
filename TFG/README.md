# Archivos para TFG
La carpeta src contiene los archivos .vhd del proyecto

## espTOOL
**espTOOL** es un generador de teselas 8x8 en formato ROM desde un esp32s3

## pyTOOL
**pyTOOL** es un generador, mediante python, del mapa de nombres y mapa de teselas de una imagen en formato .png, .jpg, .tiff en formato ROM
